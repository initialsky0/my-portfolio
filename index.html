About me

Hi, my name is Dong (initialsky0), and welcome to my portfolio. If you ask me, is there magic in this world? I’ll answer yes there is. And to me coding and programming is the spell, and hardwares are the magic staffs that make magic happen in the real world. Through a few lines of code, computers can do logical calculations for humans. And because programs can handle repeating tasks easily and quickly, people can focus on being more creative and more efficient to achieve a greater task. In the current age, many things can be done with one click or tab on tablet or pc. This is completely different compared to the past. And it may be completely different in the future as technology advances.

 

I really want to see the changes in the front line. And that's why I attended an engineering college and decided to work in an electronic retail store. But in order to really reach the front line, what I’m doing is not enough. Therefore I decided to dive into coding, and start to learn this skill which is dynamic and fascinating. During my learning adventure, I discovered many techniques people use to create modern apps. The following section contains online course projects I worked on with what skills I learned from them. Please take your time and check it out!

Facial Recognition App

Check it out

A project from an online course taught by Andrei Neagoie. This project is also built using React library. It is a full fledged App that has front-end and back-end components. The project consist of a log-in, register form, and main UI page for face detection. User data is stored on the database, and face detection is based on an API from Clarifai. The app is also deployed on Heroku.
